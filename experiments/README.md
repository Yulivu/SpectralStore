# Experiments

Each experiment lives in its own subdirectory:

```text
experiments/
  preliminary/
    bitcoin_otc/
      configs/
      results/
      README.md
  smoke/
    quickstart/
      configs/
      results/
      README.md
```

`results/` folders are ignored by Git except for `.gitkeep` placeholders.
Commit configs and README files so each experiment remains reproducible.
